# 易有云团队

目标：提供更简单易用的网络存储方案

## 官网

https://www.linkease.com

## 提供的产品

### iStoreOS

基于 OpenWRT 一个可用于路由或者网络存储场景的操作系统。

### 易有云

提供整套私有化的网络存储服务。客户端支持 iOS、Android、macOS、Windows，服务器端支持 iStoreOS、Merlin、群晖、QNAP、Windows、macOS、Linux 。

功能：

* 远程应用（远程路由，远程下载等）
* 相册备份
* 设备互联
* 文件多端同步

特性：

* 提供点对点的文件传输
* 提供增量文件传输
* 提供流媒体播放
* 传输速度极快
